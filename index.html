Sobre o Projeto

Tema: Desenvolvimento de um Website Interativo

Objetivo: Demonstrar competências técnicas na criação de soluções web modernas e acessíveis.

Ferramentas: HTML, CSS, JavaScript e design responsivo.

3. Implementação e testes de usabilidade.

4. Apresentação e defesa do trabalho perante o júri.
